## Tech Debt (non-blocking)

All items below are documented in their phase's own code review / verification and were
explicitly scoped out — none block the milestone's definition of done. Ranked by significance:

1. **[09] CR-01 — untrusted-input parity gap.** `loadManifests()`'s `dapps` and inline
   `manifests` tiers still throw an uncaught `TypeError` on a non-array value before
   `window.__DXKIT__` is exposed. ROB-05 was scoped to the `registry.json` fetch tier only, so
   this is a legitimate scope-extension, not a ROB-05 failure — but it's the one item most worth
   a follow-up given IIFE/static-HTML is the first-class deployment target. Fix: extract the
   shared `coerceManifestArray()` helper CR-01 already proposes.
2. **[07] DeepPartial/deepMerge latent gaps.** Function/Date/Map/Set/RegExp/class-instance values
   are mishandled on merge. Unreachable through `DappManifest` today (plain-JSON callers); exact
   patch documented in 07-REVIEW.
3. **[08] smoke/ not typechecked.** `smoke/` is outside `tsconfig.typecheck.json`'s include, so a
   type error there escapes `make typecheck`. Fix: add `smoke` to the include.
4. **[06] @types/node@25 vs Node 22/24 floor** (WR-03), Makefile `.PHONY` `format`/`lint-format`
   mismatch, engines formatting split, and the stale `>=22` literal in ROADMAP SC1/D-05.
5. **Minor** (09 WR-01/WR-02, and assorted IN-level findings across phases): a vacuous `||` in a
   guard test, a missing try/catch in `check-no-runtime-deps.cjs` (still fails closed), and a
   handful of cosmetic edge cases.
