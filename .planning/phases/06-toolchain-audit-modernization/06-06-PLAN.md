The **intent** ("an enforced Node 22 LTS floor") is preserved and corrected: `22.12.0` is vite@8.1.4's real floor, and Node 23.x is intentionally excluded because vitest@4.1.10 rejects it. Downstream docs and ROADMAP references to the `>=22` floor MUST be updated to the corrected string by the docs pass. This narrowing of the previously-declared support surface is a breaking change (D-10) — the engines commit carries a `BREAKING CHANGE:` footer.
